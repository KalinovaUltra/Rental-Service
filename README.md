# Rental-Service
РПИ React

<br>**Предварительные требования:**
<br>Node.js (рекомендуемая версия LTS)
<br>npm (устанавливается вместе с Node.js)
<br>Редактор VS Code
<br>Терминал (встроенный в VS Code) или командная строка


<br>**1. Создание проекта**
<br>Откройте терминал в VScode (терминал - новый терминал (command prompt)) или командную строку
<br>Перейдите в папку, где будет создан проект (создайте папку rental-service).
<br>Создайте новый проект с Vite и TypeScript
<br>npm create vite@latest client --template react-ts (на возникающие вопросы выбираем - 
<br>Select framework - React
<br>Select  variant - TypeScript
<br>Перейдите в папку проекта cd client


<br>**2. Установка зависимостей**
<br>Установим базовые зависимости npm install
<br>Установим дополнительные библиотеки npm install react-router-dom @reduxjs/toolkit react-redux
<br>Установим инструменты для линтинга и разработки npm install --save-dev eslint @typescript-eslint/eslint-plugin @typescript-eslint/parser eslint-plugin-react eslint-plugin-react-hooks

<br>**3. Для работы с Redux необходимо установить пакеты @reduxjs/toolkit (Redux Toolkit) и react-redux. Первый пакет упрощает работу с Redux, а второй позволяет использовать Redux вместе с React.**
<br>**4. Также необходимо скачать библиотеку classNames через командную строку.**
<br>**5. Запуск реакта через командную строку.**
<br>Перейдите в папку с проектом: cd "C:\...\rental-service\client"
<br>Проект будет запускается через командную строку с помощью команды npm start
<br>Ctrl + C для выхода
<br>Перейдите по адресу http://localhost:5173/ и убедитесь, что проект запущен.
